SwingFish - Rip Grooveshark

Published on February 5, 2011.

SwingFish is a tiny app with one function: to download songs from Grooveshark. Check out the screenshot and download it.

Making it

Writing SwingFish was simple:

  1. Use Wireshark to sniff the traffic.
  2. Create a stand-alone app to display Grooveshark.
  3. Add a little code to rip the songs.

That's really all there was to it. (This same technique works with Pandora too...)
